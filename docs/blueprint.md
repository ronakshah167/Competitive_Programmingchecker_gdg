# **App Name**: ALL IN

## Core Features:

- Code Evaluation: Evaluate code for correctness, efficiency, and style, providing feedback and a rating. The feedback must be friendly, easy to understand, human-like and beginner-safe
- Real-time Code Execution: Allow users to execute code snippets and view the results in real-time to aid debugging.
- Personalized Feedback with AI: Employ a tool that analyzes code submissions and generates tailored feedback using AI, adapting to the user's skill level and coding style.
- Leaderboard: Rank users based on their code evaluation scores, fostering a competitive and engaging environment.
- CP Tracker: Enable users to track their progress in competitive programming, monitoring their submissions and ratings across different platforms. All core features results must be stored in the database.
- AI Chatbot: Integrate an AI-powered chatbot to provide instant assistance with coding-related questions and offer guidance. Evaluation language must be friendly, easy to understand, human-like and beginner-safe.
- User Profile: Allow users to create and manage their profiles, showcasing their coding skills and achievements.

## Style Guidelines:

- Primary Blue: #2B6CB0 for a hero gradient.
- Light Sky Blue: #90CDF4 for accents.
- Soft Ice Blue: #E0F2FE for text.
- Deep Black: #020617 for the body background.
- Pure White: #FFFFFF for text.
- Muted Gray: #94A3B8. Only use the color tones across the entire app provided by the client.
- Headings: 'Cardo' serif
- Body text: 'Glacial Indifference' sans-serif
- Code editor: Monospace (system) Note: currently only Google Fonts are supported.
- Welcome Page: Left side with heading 'GO ALL IN ON YOUR CODE' and subheading, right side with a coding-related image. Rounded corners, no clutter.
- Navigation Bar: Sticky, includes Home, Evaluate, Leaderboard, CP Tracker, Chatbot, Profile, and visible on all pages.
- Image must be coding-related, minimal, and with neutral tones. Avoid flashy illustrations or too many colors.
